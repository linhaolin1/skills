# 财富榜单数据 HTTP API

## 概述

本文档基于 AKShare 封装好的 Python 库，提供可以通过 Python 调用的财富榜单数据接口。所有接口均来自财富中文网、胡润研究院等公开数据源。

## 重要说明

- 所有接口均为 Python 函数调用
- 返回格式为 pandas.DataFrame
- 数据来源：财富中文网、胡润研究院公开 API
- 数据仅供学术研究使用，不构成投资建议

---

## 核心接口

### 1. 财富世界500强榜单

**接口名称**: 财富世界500强排行榜

**数据源**: 财富中文网

**目标地址**: https://www.fortunechina.com/fortune500/index.htm

**描述**: 获取历年财富世界500强公司排行榜

**请求方式**: GET

**返回字段说明**:
| 字段 | 说明 |
|------|------|
| 排名 | 世界500强排名 |
| 公司名称(中文) | 公司中文名称 |
| 公司名称(英文) | 公司英文名称 |
| 营业收入(百万美元) | 年营业收入 |
| 利润(百万美元) | 年利润 |
| 国家 | 公司所属国家 |
| 行业 | 公司所属行业 |

**Python 调用示例**:

```python
import akshare as ak

# 获取财富世界500强榜单数据
df = ak.fortune_rank(year="2023")
print(df)
```

### 2. 胡润百富榜

**接口名称**: 胡润百富榜

**数据源**: 胡润研究院

**目标地址**: https://www.hurun.net/zh-CN/Rank/HsRankDetailsList

**描述**: 获取胡润百富榜数据

**请求方式**: GET

**输入参数**:
| 参数名 | 类型 | 必填 | 说明 | 示例值 |
|--------|------|------|------|--------|
| num | string | 是 | 年份代码 | 从页面获取 |
| search | string | 否 | 搜索关键词 | 空 |
| offset | string | 是 | 偏移量 | 0 |
| limit | string | 是 | 每页数量 | 20000 |

**返回字段说明**:
| 字段 | 说明 |
|------|------|
| 排名 | 榜单排名 |
| 财富 | 财富金额(亿元) |
| 姓名 | 富豪姓名 |
| 企业 | 所属企业 |
| 行业 | 所属行业 |

**Python 调用示例**:

```python
import akshare as ak

# 获取胡润百富榜数据
df = ak.hurun_rank(indicator="胡润百富榜", year="2023")
print(df)
```

## 注意事项

1. 财富榜单通常每年更新一次
2. 不同榜单的统计口径可能不同
3. 胡润榜单需要从页面获取年份代码(num参数)
4. 数据仅供学术研究，不构成投资建议
5. 财富数据为公开估算值，可能与实际有出入
