# REITs数据 HTTP API

## 概述

本文档基于 AKShare 封装好的 Python 库，提供可以通过 Python 调用的 REITs（不动产投资信托基金）数据接口。所有接口均来自东方财富网等公开数据源。

## 重要说明

- 所有接口均为 Python 函数调用
- 返回格式为 pandas.DataFrame
- 数据来源：东方财富网公开 API
- 数据仅供学术研究使用，不构成投资建议

---

## 核心接口

### 1. REITs实时行情

**接口名称**: 沪深REITs实时行情列表

**数据源**: 东方财富网

**目标地址**: https://95.push2.eastmoney.com/api/qt/clist/get

**描述**: 获取沪深REITs的实时行情数据

**请求方式**: GET

**输入参数**:
| 参数名 | 类型 | 必填 | 说明 | 示例值 |
|--------|------|------|------|--------|
| pn | string | 是 | 页码 | 1 |
| pz | string | 是 | 每页数量 | 100 |
| po | string | 是 | 排序 | 1 |
| np | string | 是 | 参数 | 1 |
| ut | string | 是 | 用户标识 | bd1d9ddb04089700cf9c27f6f7426281 |
| fltt | string | 是 | 过滤类型 | 2 |
| invt | string | 是 | 参数 | 2 |
| fid | string | 是 | 字段ID | f3 |
| fs | string | 是 | 市场筛选 | m:1 t:9 e:97,m:0 t:10 e:97 |
| fields | string | 是 | 返回字段 | f2,f3,f4,f5,f6,f12,f14,f15,f16,f17,f18 |

**返回字段说明**:
| 字段 | 说明 |
|------|------|
| 序号 | 排名序号 |
| 代码 | REITs代码 |
| 名称 | REITs名称 |
| 最新价 | 最新成交价 |
| 涨跌额 | 涨跌金额 |
| 涨跌幅 | 涨跌幅度(%) |
| 成交量 | 成交量(手) |
| 成交额 | 成交金额 |
| 开盘价 | 今日开盘价 |
| 最高价 | 今日最高价 |
| 最低价 | 今日最低价 |
| 昨收 | 昨日收盘价 |

**Python 调用示例**:

```python
import akshare as ak

# 获取沪深REITs实时行情数据
df = ak.reits_realtime_em()
print(df)
```

### 2. REITs历史K线数据

**接口名称**: REITs历史行情

**数据源**: 东方财富网

**目标地址**: https://push2his.eastmoney.com/api/qt/stock/kline/get

**描述**: 获取REITs的历史K线数据

**请求方式**: GET

**输入参数**:
| 参数名 | 类型 | 必填 | 说明 | 示例值 |
|--------|------|------|------|--------|
| secid | string | 是 | 证券ID | 市场代码.REITs代码，如：1.508097 |
| klt | string | 是 | K线类型 | 101(日线) |
| fqt | string | 是 | 复权类型 | 1 |
| lmt | string | 是 | 限制条数 | 10000 |
| end | string | 是 | 结束日期 | 20500000 |
| iscca | string | 是 | 参数 | 1 |
| fields1 | string | 是 | 字段1 | f1,f2,f3,f4,f5,f6,f7,f8 |
| fields2 | string | 是 | 字段2 | f51,f52,f53,f54,f55,f56,f57,f58,f59,f60,f61,f62,f63,f64 |
| ut | string | 是 | 用户标识 | f057cbcbce2a86e2866ab8877db1d059 |
| forcect | string | 是 | 参数 | 1 |

**返回字段说明**:
| 字段 | 说明 |
|------|------|
| 日期 | 交易日期 |
| 今开 | 今日开盘价 |
| 最高 | 最高价 |
| 最低 | 最低价 |
| 最新价 | 收盘价 |
| 成交量 | 成交量 |
| 成交额 | 成交金额 |
| 振幅 | 振幅(%) |
| 换手 | 换手率(%) |

**Python 调用示例**:

```python
import akshare as ak

# 获取REITs历史K线数据
df = ak.reits_hist_em(symbol="508097")
print(df)
```

## REITs基础知识

### 什么是REITs

REITs（Real Estate Investment Trusts）即不动产投资信托基金，是一种通过发行收益凭证募集资金，由专门投资机构进行不动产投资经营管理，并将投资综合收益按比例分配给投资者的一种信托基金。

### REITs分类

| 类型 | 说明 |
|------|------|
| 产权类 | 持有不动产产权，主要收益来源为租金和资产增值 |
| 经营权类 | 拥有不动产经营权，主要收益来源为经营收入 |

### 底层资产类型

| 类型 | 说明 |
|------|------|
| 产业园区 | 科技园区、工业园区等 |
| 仓储物流 | 物流仓库、配送中心等 |
| 高速公路 | 收费公路、桥梁等 |
| 生态环保 | 污水处理、垃圾焚烧等 |
| 保障性租赁住房 | 长租公寓、保障房等 |

## 注意事项

1. REITs交易时间与股票相同
2. REITs实行T+1交易制度
3. REITs有强制分红要求（通常90%以上可分配利润）
4. 数据仅供学术研究，不构成投资建议
5. REITs价格受底层资产经营状况影响
