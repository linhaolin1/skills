# 加密货币数据 HTTP API

## 概述

本文档基于 AKShare 实际使用的数据源 API，提供可以直接通过 curl 调用的加密货币数据接口。所有接口均来自金十数据等公开数据源。

## 重要说明

- 所有接口均为 HTTP GET 请求
- 返回格式为 JSON
- 数据来源：金十数据等公开 API
- 数据仅供学术研究使用，不构成投资建议

---

## 核心接口

### 1. 比特币持仓报告

**接口名称**: 比特币持仓报告

**数据源**: 金十数据

**目标地址**: https://datacenter-api.jin10.com/bitcoin_treasuries/list

**描述**: 获取持有比特币的上市公司持仓数据

**请求方式**: GET

**请求头**:
| 参数名 | 说明 | 示例值 |
|--------|------|--------|
| X-App-Id | 应用ID | lnFP5lxse24wPgtY |
| X-Version | 版本号 | 1.0.0 |

**返回字段说明**:
| 字段 | 说明 |
|------|------|
| 代码 | 公司代码 |
| 公司名称-英文 | 公司英文名 |
| 公司名称-中文 | 公司中文名 |
| 国家/地区 | 所在国家或地区 |
| 市值 | 公司市值 |
| 比特币占市值比重 | 比特币持仓占市值比例 |
| 持仓成本 | 持仓成本 |
| 持仓占比 | 持仓占比 |
| 持仓量 | 比特币持仓数量 |
| 当日持仓市值 | 持仓市值 |
| 查询日期 | 数据日期 |
| 公告链接 | 公告链接 |
| 分类 | 公司分类 |

**curl 调用示例**:

```bash
# 获取比特币持仓报告
curl -s "https://datacenter-api.jin10.com/bitcoin_treasuries/list" \
  -H "X-App-Id: lnFP5lxse24wPgtY" \
  -H "X-Version: 1.0.0"
```

### 2. 加密货币实时行情

**接口名称**: 加密货币行情

**数据源**: 各大交易所

**目标地址**: 各交易所公开API

**描述**: 获取加密货币实时价格

**请求方式**: GET

**curl 调用示例**:

```bash
# 获取比特币价格（示例，需要具体交易所API）
# Binance API
curl -s "https://api.binance.com/api/v3/ticker/price?symbol=BTCUSDT"

# 获取以太坊价格
curl -s "https://api.binance.com/api/v3/ticker/price?symbol=ETHUSDT"
```
